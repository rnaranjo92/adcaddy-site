Platform Integrations

// SECTION_002

TikTok Integration

ACTIVE

Ad Caddy integrates with TikTok to allow content creators to publish short-form video content directly from the desktop app. This eliminates the need to manually upload videos through the TikTok mobile app, streamlining the content workflow for businesses managing multiple social platforms.

Scopes used: user.info.basic (to identify the connected account), video.publish (to upload videos via the Content Posting API), and video.upload (for the pull-from-URL upload flow).
How it works: Users authenticate via OAuth, select TikTok as a target platform in the Deploy page, attach a video file, write a caption, and click Launch. Ad Caddy uploads the video to TikTok using the Content Posting API and tracks the publish status.
